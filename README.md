# CloudWatch Costs Dashboard

This repository stores the CloudFormation template for the "CloudWatch-Costs" dashboard used to visualize Amazon CloudWatch costs.

Original AWS blog post:
- https://aws.amazon.com/blogs/mt/visualizing-amazon-cloudwatch-costs-part-1/

Original template (reference only):
- https://d2908q01vomqb2.cloudfront.net/artifacts/MTBlog/cloudops-927/cloudformation-usage-cloudwatchv2.yaml

## Stack

- CloudFormation stack name: DEVOPS-Cloudwatch-Monitoring-Git
- Purpose: Show CloudWatch usage and costs (logs ingestion, storage, metrics API calls).
- Managed via: AWS CloudFormation Git sync.
